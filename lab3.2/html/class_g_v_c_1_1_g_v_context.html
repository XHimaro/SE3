GVContext Create a graph context with built-in plugins and optionally load dynamic plugins on demand.

Аргументы
builtinsArray of structs, each containing a pointer to the name and the address of a plugin to load. The end of the array is designated by a struct whose name pointer is NULL.
demand_loadingEnables loading plugins dynamically on demand when true.
