\doxysubsection{Подробное описание}
Header of a user record. 

These records are attached by client programs dynamically at runtime. \mbox{\hyperlink{class_a}{A}} unique string ID must be given to each record attached to the same object. Cgraph has functions to create, search for, and delete these records. The records are maintained in a circular list, with obj-\/\texorpdfstring{$>$}{>}data pointing somewhere in the list. The search function has an option to lock this pointer on a given record. The application must be written so only one such lock is outstanding at a time.
