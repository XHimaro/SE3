41/* The -T output formats listed below are examples only, they are not definitive or inclusive,
42 other outputs may use the flags now, or in the future
43
44 Default emit order is breadth first graph walk order
45 EMIT_SORTED emits nodes before edges
46 EMIT_COLORS emits colors before nodes or edge -Tfig
47 EMIT_CLUSTERS_LAST emits cluster after nodes and edges
48 EMIT_PREORDER emit in preorder traversal ???
49 EMIT_EDGE_SORTED emits edges before nodes
50
51 GVDEVICE_DOES_PAGES provides pagination support -Tps
52 GVDEVICE_DOES_LAYERS provides support for layers -Tps
53 GVDEVICE_EVENTS supports mouse events -Tgtk, -Txlib
54 GVDEVICE_DOES_TRUECOLOR supports alpha channel -Tpng, -Tgtk, -Txlib
55 GVDEVICE_BINARY_FORMAT Suppresses \r\n substitution for linends
56 GVDEVICE_COMPRESSED_FORMAT controls libz compression
57 GVDEVICE_NO_WRITER used when gvdevice is not used because device uses its own writer, devil outputs (FIXME seems to overlap OUTPUT_NOT_REQUIRED)
58
59 GVRENDER_Y_GOES_DOWN device origin top left, y goes down, otherwise
60 device origin lower left, y goes up
61 GVRENDER_DOES_TRANSFORM device uses scale, translate, rotate to do its own
62 coordinate transformations, otherwise coordinates
63 are pre-transformed
64 GVRENDER_DOES_ARROWS renderer has its own idea of arrow shapes (deprecated)
65 GVRENDER_DOES_LABELS basically, maps don't need labels
66 GVRENDER_DOES_MAPS renderer encodes mapping information for mouse events -Tcmapx -Tsvg
67 GVRENDER_DOES_MAP_RECTANGLE supports a 2 coord rectngle optimization
68 GVRENDER_DOES_MAP_CIRCLE supports a 1 coord + radius circle optimization
69 GVRENDER_DOES_MAP_POLYGON supports polygons (basically, -Tsvg uses anchors, so doesn't need to support any map shapes)
70 GVRENDER_DOES_MAP_ELLIPSE supports a 2 coord ellipse optimization
71 GVRENDER_DOES_MAP_BSPLINE supports mapping of splines
72 GVRENDER_DOES_TOOLTIPS can represent tooltip info -Tcmapx, -Tsvg
73 GVRENDER_DOES_TARGETS can represent target info (open link in a new tab or window)
74 GVRENDER_DOES_Z render support 2.5D representation -Tvrml
75 GVRENDER_NO_WHITE_BG don't paint white background, assumes white paper -Tps
76 LAYOUT_NOT_REQUIRED don't perform layout -Tcanon
77 OUTPUT_NOT_REQUIRED don't use gvdevice for output (basically when agwrite() used instead) -Tcanon, -Txdot
78 */
