26/* Type indicating granularity and method
27 * l_undef - unspecified
28 * l_node - polyomino using nodes and edges

30 * (assumes ND_clust(n) unused by application)
31 * l_graph - polyomino using computer graph bounding box
32 * l_array - array based on graph bounding boxes
33 * l_aspect - tiling based on graph bounding boxes preserving aspect ratio
34 * l_hull - polyomino using convex hull (unimplemented)

36 * l_bisect - alternate bisection using graph bounding box (unimplemented)
