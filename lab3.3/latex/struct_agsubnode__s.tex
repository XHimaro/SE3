\doxysubsection{Подробное описание}
This is the node struct allocated per graph (or subgraph). 

It resides in the n\+\_\+dict of the graph. The node set is maintained by libdict, but transparently to libgraph callers. Every node may be given an optional string name at its time of creation, or it is permissible to pass NIL(char$\ast$) for the name.
